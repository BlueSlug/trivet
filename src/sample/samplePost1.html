Creating sites with blog posts or news articles

29th July 2020 â€” 1 minute read

Follow the follwoing steps to create a blog based fluidic site based on 11ty.

  1. Go to the fluidic-11ty repository. Fork it and clone the fork to your local.


  2. Remove the src/sample/ directory.
  3. Remove the src/assets/sampleImages folder too.
  4. Remove the contents of src/assets/stylesheets/css/sample folder
  5. The templates for the site lie in the src/_includes folder
  6. Copy the contents of the header HTML code to the src/_includes/partials/header.html file.
  7. Copy the contents of footer file to the src/_includes/partials/footer.html file
  8. Copy the CSS files to src/assets/stylesheets/css folder
  9. Now, you can create the index page for your blog site by editing the src/index.html file
  10. Next, you need to link the CSS files to Default and Post template. Go to src/_includes/partials/foundationflexgrid.html file and link them as per sample links given in the file
  11. Now in order to create blog posts, go to src/blogs folder and create blogs using the template file present in the Blogs folder
  12. You can store all the images related to blog files in src/blogs/images folder.
